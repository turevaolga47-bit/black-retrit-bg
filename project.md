# project.md — Чёрный Ретрит Болгария

## Что это за проект
Лендинг для женского ретрита у Чёрного моря в Болгарии. Автор и ведущая — Ph.Dr. Ольга Турьева, психолог. Цель сайта — привлечь участниц и принять заявки через Telegram-бота.

## Для кого
Женщины, которые чувствуют усталость, потерю себя, хотят внутренней перезагрузки и трансформации в тёплой, поддерживающей обстановке.

## Страницы и разделы (уже есть в landing.html)
| Раздел | Описание |
|--------|----------|
| Hero | Заголовок, дата, CTA-кнопка |
| Для кого | Описание целевой аудитории |
| Галерея | Фото места |
| Три столпа трансформации | Ключевые блоки программы |
| 10 дней | Подробности программы |
| Бонусы | Что входит дополнительно |
| Об Ольге | Биография и дипломы |
| FAQ | Ответы на частые вопросы |
| Инфокарточки | Краткие факты о ретрите |
| CTA | Финальный призыв к действию |

## Дизайн
- Фон: очень тёмный, почти чёрный (`#0a0a0f`) с эффектом звёздного неба
- Текст: светлый кремовый (`#e8e0d5`)
- Акценты: золотой (`#c9a84c`) и розовый (`#d4829a`)
- Шрифт: Georgia (с засечками, элегантный)
- Стиль: ночной, мистический, тёплый

## Технический стек
- Один файл: `landing.html` (HTML + CSS + JS внутри)
- Без внешних библиотек и фреймворков
- Telegram-бот: `bot.js` (Node.js, запуск `node bot.js`)
- Деплой: GitHub Pages
